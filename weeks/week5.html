What You'll Build
30-Day Final Capstone Project
Production AI Copilot: Full-Stack Architecture to Deployment
The capstone integrates every major concept from the 30-day curriculum into a single, shipable AI Copilot system. You will design, build, test, and deploy a production-grade multi-agent system that a Senior/Staff AI Engineer would be proud to put on a portfolio.
  • Copilot architecture diagram: context window manager, short/long-term memory tiers, tool router
  • Multi-agent backend: LangGraph orchestrator + specialist agents (Code, Docs, Search, Critic)
  • RAG pipeline: LangChain LCEL + pgvector with HyDE retrieval and cross-encoder reranking
  • Fine-tuned domain adapter (LoRA) loaded via HuggingFace PEFT for domain-specific responses
  • FastAPI backend with SSE streaming, JWT multi-tenant auth, and Pydantic structured outputs
  • NeMo Guardrails layer blocking unsafe prompts with audit logging to MLflow
  • RAGAS evaluation suite + LangSmith traces providing real-time quality monitoring
  • Docker image + Kubernetes Helm chart deployed to AWS EKS with GPU node pool and HPA
  • GitHub Actions CI/CD: lint → test → eval → deploy → smoke test on every push to main
