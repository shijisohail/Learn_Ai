What You'll Build
Week 3 Capstone Project
Fine-Tuned Domain Adapter with Full MLOps Observability
By the end of Week 3, you will have fine-tuned a base language model for a domain-specific task, tracked all experiments in MLflow, deployed the model through an MLOps pipeline, and instrumented the system with RAGAS-based evaluation and LangSmith tracing.
  • PyTorch inference harness with half-precision and flash-attention optimizations
  • QLoRA fine-tune of a 7B model on domain data using HuggingFace PEFT + TRL
  • MLflow experiment tracking with hyperparameter sweeps and artifact storage
  • Model registry promotion flow: development → staging → production
  • CI/CD pipeline (GitHub Actions) triggering automated evaluation on every merge
  • RAGAS + LangSmith dashboard monitoring faithfulness, precision, and recall in production
