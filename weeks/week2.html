Week 2 Capstone Project
Multi-Provider RAG Pipeline with CrewAI Agents
By the end of Week 2, you will have built a production-ready RAG system that combines multiple LLM providers, a vector database backend, and a CrewAI agent crew for specialized retrieval and synthesis tasks.
  • LangChain LCEL pipeline wiring retriever → reranker → LLM with streaming output
  • Vector database with pgvector or Pinecone storing 1,000+ chunked documents
  • Multi-provider fallback: Claude primary, GPT-4o fallback, Gemini for embeddings
  • CrewAI crew with Researcher, Synthesizer, and Critic roles collaborating on answers
  • HyDE + multi-query retrieval boosting recall by ~30% over naive top-k search
  • FastAPI endpoint exposing the pipeline with SSE streaming responses
