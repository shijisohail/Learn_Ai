🔌 Local server not running

The AI Coach needs a local proxy server. Add your OpenRouter API key to .env and start it:

# Install dependencies (first time only)
pip install fastapi uvicorn httpx python-dotenv

# Add your key to .env
OPENROUTER_API_KEY=sk-or-…

# Start the server
python server.py

The server runs at http://localhost:8000 and streams responses directly to this page — your API key never leaves your machine.
