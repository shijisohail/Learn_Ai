@dsl.component vs @dsl.pipeline: A component is a single containerized step (runs in its own pod). A pipeline is a DAG of components. The compiler serializes the DAG to a YAML spec that Kubeflow Pipelines can execute on any Kubernetes cluster.

CI/CD Gate for Model Promotion

Write a promotion script that only upgrades the production model if evaluation metrics pass threshold checks.

Completed
Your Task

Build a quality gate for automated model promotion:

  • Write a evaluate_candidate(run_id) function that reads metrics from MLflow
  • Define threshold checks: val_loss < 1.5, ragas_score > 0.7
  • Only promote to Production if ALL checks pass
  • Log promotion decisions as MLflow tags on the run
  • Return a structured result: {"promoted": bool, "checks": {...}, "reason": str}
