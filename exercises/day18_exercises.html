Completed
Your Task

Log multiple artifact types to an MLflow run:

LangSmith Tracing for LLM Runs

Automatically trace every LLM call, chain, and RAG pipeline to LangSmith for debugging and evaluation.

Completed
Your Task

Instrument LLM calls with LangSmith tracing:

  • Set LANGCHAIN_TRACING_V2=true and LANGCHAIN_API_KEY env vars
  • Use the @traceable decorator from langsmith to trace any Python function
  • Pass metadata like run_type="llm" and name="my-rag-pipeline"
  • Wrap an Ollama call in a traceable function and verify the trace appears in the LangSmith UI
  • Use langsmith.Client() to list recent runs from the project
