FastAPI Inference Service

Serve the merged model behind a FastAPI endpoint with Prometheus metrics and async generation support.

Completed
Your Task

Build the inference API with full observability:

  • Load ./capstone_model in @app.on_event("startup") to warm the model once
  • Define POST /generate accepting {"prompt": str, "max_tokens": int}
  • Format the prompt using tokenizer.apply_chat_template() before generating
  • Instrument with Prometheus: request counter (label: status), latency histogram, tokens/sec gauge
  • Add GET /health returning model name, uptime, and total requests served
