Embedding Drift Detection with scipy

Detect when production queries drift away from your training distribution using cosine similarity and KS-test.

Completed
Your Task

Implement a drift detector for query embeddings:

  • Create a DriftDetector class that stores a reference embedding distribution
  • Compute pairwise cosine similarity between reference and production batches using scipy.spatial.distance.cdist

  • Flag drift if the KS test p-value is below 0.05
  • Return a report: {"drift_detected": bool, "ks_statistic": float, "p_value": float, "mean_similarity": float}
