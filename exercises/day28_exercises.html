Exercise 04

Self-Improvement Loop

Generate multiple responses, score them with an LLM judge, and collect the top-k as new training examples automatically.

Objective

Implement a RLAIF-style self-improvement cycle: sample N responses, score each with a judge, use top-k as chosen and bottom-k as rejected for a new preference dataset.

  • Implement sample_responses(prompt, n=4, temperature=0.8) returning N diverse responses
  • Implement score_response(prompt, response) returning a float 0-10 from the judge
  • Implement build_preference_pairs(prompt, scored_responses) pairing top-k vs bottom-k
  • Run the loop on 3 different prompts and collect all pairs
  • Print a summary showing the score gap between chosen and rejected responses
