Your Task

Implement require_scope():

  • Get the user context: user = current_user.get() — raises LookupError if not set
  • If scope not in user.scopes: raise PermissionError(f"Missing scope: {scope}")
  • Otherwise: call and return fn(*args, **kwargs)
  • Write a test that sets current_user.set(UserContext(...)) before calling the decorated tool
  • Verify that async tools (with async def) also work by using asyncio.iscoroutinefunction check

Add structured JSON audit logs to every AI tool call, capturing who called what, when, with what args, and what the outcome was.

# pip install structlog import structlog, time, json from functools import wraps from contextvars import ContextVar audit_log = structlog.get_logger("audit") current_user: ContextVar = ContextVar("current_user", default=None) def audit_tool(fn): """Decorator: log every tool call with user context, args, outcome.""" @wraps(fn) def wrapper(*args, **kwargs): user = current_user.get() start = time.perf_counter() try: result = fn(*args, **kwargs) # TODO: log success with user context, tool name, args, duration_ms return result except Exception as e: # TODO: log failure with error type and message raise return wrapper
Your Task

Complete @audit_tool:

  • On success: call audit_log.info("tool_call", user_id=user.user_id, tenant_id=user.tenant_id, tool=fn.__name__, kwargs=kwargs, outcome="success", duration_ms=int((time.perf_counter()-start)*1000))
  • On exception: log with outcome="error", error_type=type(e).__name__, error_msg=str(e), then re-raise
  • Configure structlog to output JSON: structlog.configure(processors=[..., structlog.dev.JSONRenderer()])
  • Combine with @require_scope from Exercise 3 — stack both decorators
