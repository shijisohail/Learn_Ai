LoRA math: For a weight matrix W∈ℝ^(d×k), LoRA adds B∈ℝ^(d×r) and A∈ℝ^(r×k) where r≪min(d,k). Only BA is trained — about (d+k)r parameters vs d*k. For GPT-2's c_attn (768×2304), r=16 gives 2.5% trainable params.

Merge trained LoRA weights back into the base model to get a single self-contained model for deployment.

Completed
Your Task

Why merge? Keeping the PEFT adapter separate adds runtime overhead (extra matrix multiply). After merging, the effective weight is W' = W + BA·(alpha/r), stored directly — zero inference overhead and compatible with vLLM, llama.cpp, etc.
